# Sandesh Kumar Portfolio

One Piece inspired personal portfolio website built around a weathered pirate-paper UI, wanted bounty poster layout, and clean developer resume presentation.

Live site: [sandesh-portfolio-seven.vercel.app](https://sandesh-portfolio-seven.vercel.app/)

## Overview

This portfolio presents my work as a full-stack developer and AI/ML engineer through a custom One Piece themed experience. The design uses parchment textures, sepia tones, bold wanted-poster typography, interactive profile imagery, project log cards, and a themed contact form.

## Features

- Split-screen landing section with looping intro video
- Wanted poster profile section with hover image flip
- KIIT-issued academic and resume-style experience logs
- Clean tech stack section without fake skill percentages
- Featured GitHub project cards with direct repository links
- One Piece styled query/contact form
- Custom AZ logo and favicon
- Fully responsive layout for desktop and mobile

## Local Setup

Clone the repository:

```bash
git clone https://github.com/Sandeshkumar0/Sandesh_portfolio.git
cd Sandesh_portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the local app:

```text
http://localhost:3000
```

## Build

Create a production build:

```bash
npm run build
```

Run lint checks:

```bash
npm run lint
```
