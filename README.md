# Sandesh Portfolio

Personal portfolio website built with Next.js, React, Framer Motion, GSAP, and Tailwind CSS.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

Create a production static export:

```bash
npm run build
```

## Deployment

This repository includes a GitHub Actions workflow that deploys the static Next.js export to GitHub Pages on every push to `main`.
